# Company Dashboard App

Multi-tenant SaaS chat application dashboard built with Next.js App Router.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Zustand
- Axios

## Project Structure
- `src/app` App Router routes
- `src/components` UI, layout, dashboard, and chat components
- `src/features` Feature hooks and logic
- `src/services` API clients and domain services
- `src/store` Zustand stores
- `src/styles` Global styles

## Scripts
- `npm run dev` Start development server
- `npm run build` Build for production
- `npm run start` Run production build
- `npm run lint` Lint

## Environment
Copy `.env.example` and set your values before running locally.

## Notes
If you change dependencies, re-run `npm install` to refresh `package-lock.json`.
